# Student Management System (Java)

## 📌 Project Description
This is a simple Student Management System built using Core Java.
It allows users to perform basic operations like adding, viewing, updating, and deleting student records.

## 🚀 Features
- Add Student
- View Students
- Update Student
- Delete Student

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- Encapsulation
- Classes & Objects
- ArrayList (Collections)
- Java Basics

## ▶️ How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run `Main.java`

## 📁 Project Structure
src/ 
|- Main.java 
|- Student.java
|- StudentService.java
